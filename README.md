# c2heung_sidequest_W6

Simple browser game built with HTML5 Canvas.

## How to run

Open `index.html` in your browser.

## Controls

- Move platform left: Left Arrow or **W**
- Move platform right: Right Arrow or **D**
- Start / restart: **N**

## Goal

- Keep the ball bouncing on the platform.
- Collect stars to increase your score.
- Ball changes color when it hits a wall.
- Game ends when the ball falls below the platform and hits the floor.
